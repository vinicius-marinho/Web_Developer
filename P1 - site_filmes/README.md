# Projeto 1

Esse projeto visa a construção de um código server-side de uma página web, que contem todos os meus filmes favoritos, seus posters e, ao clicar no poster, o trailer é exibido. Toda a linguagem server-side é construída em **python 3**.

## Usando

É necessário fazer o clone do repositório e executar o arquivo entertainment.py

```
git clone https://github.com/vinicius-marinho/Web_Developer/tree/master/P1%20-%20site_filmes
cd P1 - site_filmes
python3 entertainment.py
```
